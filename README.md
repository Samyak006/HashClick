# HashClick

## Abstract
The Idea of HashClick is to provide an interface to the user while pasting their information. They will have a memory that consists upto "n" indices of previously copied items. 

## Usability
Have a track of what information you had captured, paste previously copied items on the clipboard. 

## How will this be implemented?
the user shall have an option in the paste option where they get a history of clipboard.

## What all API's might be needed. 
 - events from the system: clipboard API if available.
 - Manually create a service that creates a database for the options. 
 - make changes to the right click option. 
 - have a temporary database in the system or on cloud to have interoperability.
